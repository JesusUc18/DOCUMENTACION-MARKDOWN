### Parte 1: Dibujos

**Descripción del proceso para que se renderizara su matriz:**

```
Abrir excel -->
Nueva hoja blanca -->
selecciono una cuadricula de 30 x 30 (A1: AD30) -->
Resduzco el tamaño (Quedan cuadrados) -->
Formato condicional -->
Añado nueva regla -->
Damos formato a las celdas (número) -->
0 == Blanco y 1 == Negro -->
Confirmamos.
```
